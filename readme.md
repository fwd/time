> A NodeJS package to simplify milliseconds.

## Install

```sh
npm install @fwd/time
```

## Usage

```js

const time = require('@fwd/time')

time(1, 'second') // 1000 ms

time(5, 'hours') // 18000000 ms

```

## Parameters

```js

time(integer, timeInterval)

````

## Available Intervals

- second(s)
- minute(s)
- hour(s)
- month(s)
- year(s)
